# App Favicon Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the default Vite favicon with a custom SVG clock-face character that matches the app's visual style.

**Architecture:** Create a single `public/favicon.svg` with a hand-drawn-style clock face featuring dot eyes, a smile, and red/blue hands. Update `index.html` to reference it. No JS changes needed — pure static assets.

**Tech Stack:** SVG, HTML

---

### Task 1: Create the favicon SVG

**Files:**
- Create: `public/favicon.svg`
- Delete: `public/vite.svg` (no longer needed)

**Step 1: Create `public/favicon.svg`**

Write the following file exactly:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <title>Clock Learning</title>

  <!-- Clock face -->
  <circle cx="50" cy="50" r="46" fill="#fffde7" stroke="#4e342e" stroke-width="5"/>

  <!-- Tick marks at each hour -->
  <line x1="50" y1="6"  x2="50" y2="13" stroke="#8d6e63" stroke-width="2.5" stroke-linecap="round"/>
  <line x1="50" y1="87" x2="50" y2="94" stroke="#8d6e63" stroke-width="2.5" stroke-linecap="round"/>
  <line x1="6"  y1="50" x2="13" y2="50" stroke="#8d6e63" stroke-width="2.5" stroke-linecap="round"/>
  <line x1="87" y1="50" x2="94" y2="50" stroke="#8d6e63" stroke-width="2.5" stroke-linecap="round"/>

  <!-- Eyes (at ~10 and ~2 o'clock positions) -->
  <circle cx="34" cy="38" r="5" fill="#4e342e"/>
  <circle cx="66" cy="38" r="5" fill="#4e342e"/>

  <!-- Smile -->
  <path d="M 36 62 Q 50 74 64 62" fill="none" stroke="#4e342e" stroke-width="3.5" stroke-linecap="round"/>

  <!-- Hour hand (pointing toward ~10) -->
  <line x1="50" y1="50" x2="31" y2="33" stroke="#ef5350" stroke-width="5" stroke-linecap="round"/>

  <!-- Minute hand (pointing toward ~2) -->
  <line x1="50" y1="50" x2="67" y2="20" stroke="#42a5f5" stroke-width="3.5" stroke-linecap="round"/>

  <!-- Centre dot -->
  <circle cx="50" cy="50" r="4" fill="#4e342e"/>
</svg>
```

**Step 2: Update `index.html`**

Change line 4 from:
```html
<link rel="icon" type="image/svg+xml" href="/vite.svg" />
```
to:
```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
```

**Step 3: Delete the old icon**

```bash
rm public/vite.svg
```

**Step 4: Verify in browser**

```bash
npm run dev
```

Open the app and confirm the browser tab shows the clock character favicon. Check at 16×16 and 32×32 sizes (zoom in on the tab).

**Step 5: Commit**

```bash
git add public/favicon.svg index.html
git rm public/vite.svg
git commit -m "feat: add custom clock character favicon"
```
